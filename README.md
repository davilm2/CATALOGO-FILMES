# CatalogoFilmes
É um catálogo simples de filmes que foi trabalho com a framework Flask. No código iremos ver bastamente Python, HTML e CSS, e um pouco de Javascript. É também utilizado um sistema de banco de dados pelo MySQL, dentro do código há também um sistema de login e logout em que só poderá adicionar novos filmes, editá-los ou apagá-los caso esteja devidamente logado no site.
